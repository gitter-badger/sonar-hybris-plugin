This rule detects direct usage of Services in Controller. In Hybris Controller should only interact with Facades and DTO.

Noncompliant Code Example


@Controller
public class HybrisControllerCheckModelUsage {
...
    CatalogService CatalogService; // Noncompliant
...
}

Compliant Solution

@Controller
public class HybrisControllerCheckModelUsage {
...
    CatalogFacade catalogFacade; // Compliant
}
