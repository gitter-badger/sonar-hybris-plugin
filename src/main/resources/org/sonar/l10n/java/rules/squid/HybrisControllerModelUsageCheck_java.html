This rule detects direct usage of Models in Controller. In Hybris Controller should only interact with Facades and DTO.

Noncompliant Code Example


@Controller
public class HybrisControllerCheckModelUsage {
...
    void nonCompliantMethod() {
    CatalogModel localCatalog; // Noncompliant
    ...
    }

}

Compliant Solution

@Controller
public class HybrisControllerCheckModelUsage {
...
    void nonCompliantMethod() {
    CatalogDTO localCatalog; // Compliant
    ...
    }

}
